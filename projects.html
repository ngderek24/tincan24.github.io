inKitchen

HackSC 2014, November 2014
Languages: HTML, CSS, Javascript

inKitchen is a web application that takes in the user's available food ingredients and generates a list of recipes corresponding to those ingredients. Our 4-member team designed this web application with the idea that college students might be able to use it for leftover food in the refrigerator. I mainly implemented the user interface and some Javascript for HTML content effects.

Password Manager

Summer 2015
Languages: Java

Password Manager is a desktop application built with Eclipse Windows Builder and written in Java. It allows the user to insert, delete, update, and search for his/her usernames and passwords with the easy-to-use tabbed interface. The information is kept through SQLite Manager. Steganography

Summer 2015
Languages: Java

Steganography is a desktop application built with Eclipse Windows Builder. It allows the user to hide secret messages within an png image and also retrieve hidden messages and output them to a text file. This technique is done through bit manipulation of the RGB values of the pixels in the image.

Movie Database Search

CS 143: Database Systems, Spring 2015
Languages: C++, PHP

My partner and I implemented a B+ tree index for a movie database as a class project for CS143 at UCLA. This index supports loading tuples from input files, "select" queries on attributes with conditional "where" clauses. Insert functions are implemented recursively in order to handle node overflows at both leaf and non-leaf levels. It is a generic template that works for any node of size less than or equal to 81 pointers. We also created the UI for inserting/searching movies/actors.

Micromouse

October 2014 - May 2015
Languages: C++ (Arduino)

Micromouse is an autonomous maze-solving robot competition. The mouse at one corner of a 16x16 maze, where it is allowed to explore the maze using IR sensors. The mouse then finds the shortest path to the center of the maze. On the hardware side, the mouse involves designing the circuit, soldering elements, and calibrating sensors. As for the software, we utilize physical components, optimize data structures, and implement maze-solving algorithms.

Aria

LA HACKS 2014, April 2014
Languages: HTML, CSS, Javascript

Aria is an web application that plays sounds and displays images based on the mood of the user. There is also a textbox for users to write about their problems and delete it once they are done as a way of relieving their stress. This was the first web app that my colleagues and I made, and I learned how to incorporate sound and images into the webpage.

BugBlast

CS 32: Introduction to Computer Science II, Winter 2014
Languages: C++

BugBlast is an arcade monster-killing game where the player has to defeat monsters using poisonous sprays, given as a project in CS32 at UCLA. I implemented the gameplay code using a given framework and specifications from the professor. I applied core concepts of C++, such as using depth-first search algorithm to create effective AI, using inheritance and polymorphism to implement object-oriented programming, and managing memory-allocation and STL tools.
