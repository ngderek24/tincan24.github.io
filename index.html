Movie Database Search

CS 143: Database Systems, Spring 2015
Languages: C++, PHP

My partner and I implemented a B+ tree index for a movie database as a class project for CS143 at UCLA. This index supports loading tuples from input files, "select" queries on attributes with conditional "where" clauses. Insert functions are implemented recursively in order to handle node overflows at both leaf and non-leaf levels. It is a generic template that works for any node of size less than or equal to 81 pointers. We also created the UI for inserting/searching movies/actors.

Micromouse

October 2014 - May 2015
Languages: C++ (Arduino)

Micromouse is an autonomous maze-solving robot competition. The mouse at one corner of a 16x16 maze, where it is allowed to explore the maze using IR sensors. The mouse then finds the shortest path to the center of the maze. On the hardware side, the mouse involves designing the circuit, soldering elements, and calibrating sensors. As for the software, we utilize physical components, optimize data structures, and implement maze-solving algorithms.
